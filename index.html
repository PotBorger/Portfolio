TreeTableApp, built with Python and the Textual library, is a terminal-based tool for rendering hierarchical JSON data. It displays the structure in a tree format and provides a tabular view of the node data. The app was initially built for displaying supercomputers' data at the Scientific Computing and Imaging Institute at the University of Utah.
